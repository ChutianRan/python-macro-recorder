# Python 鼠标键盘宏

一个简单的本地鼠标键盘宏工具，支持录制输入、保存记录、运行一次和无限重复运行。

## 安装

```powershell
python -m pip install -r requirements.txt
```

## 启动

```powershell
python macro_recorder.py
```

## 使用

- 点击 `开始记录` 或按 `F9` 开始录制。
- 执行需要记录的鼠标和键盘操作。
- 点击 `完成记录` 或按 `F10` 停止录制并保存。
- 在列表中选择记录后，可以点击 `运行一次` 或 `无限重复`。
- 选择记录后点击 `查看/编辑记录`，可以查看并修改保存下来的动作模块。
- 选择记录后点击 `重命名选中记录`，或双击记录，可以修改保存名称。
- 运行中按 `F12` 或点击 `停止运行` 可以停止宏。

## 编辑记录

编辑窗口会把原始输入拆成更容易修改的模块：

- `等待`：这段时间没有任何操作，可以修改持续秒数。
- `按住按键`：一次按下到松开的按键动作，可以修改按键和持续秒数。
- `按键事件`：单独的按下或松开事件。
- `鼠标点击`：一次鼠标按下到松开，可以修改按钮、位置和持续秒数。
- `鼠标移动` / `鼠标滚轮`：可以修改坐标和滚动量。

修改字段后先点击 `应用到模块`，确认列表变化后点击 `保存到记录`。

录制时，本软件窗口内部的鼠标操作会被忽略，避免控制按钮本身被录进宏里。控制热键 `F9`、`F10`、`F12` 也不会保存到宏记录中。

记录会保存到同目录下的 `macro_records.json`。
